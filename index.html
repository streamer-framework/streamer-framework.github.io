STREAMER

Data Streams

STREAMER is a machine learning experimenter with a realistic operational stream environment and visualization capabilities. Oriented to Data Scientists, it facilitates the integration and testing of machine learning algorithms in streaming operational contexts.

STREAMER is the first open-source academic framework that is scalable and cross-platform.

Architecture

5 modules:

  1. M1- Data Stream Ingestion
  2. M2- DSPE Processor
  3. M3- Learning API + Algs. Adapter.
  4. M4- Application & Evaluation.
  5. M5- Data Visualization.
