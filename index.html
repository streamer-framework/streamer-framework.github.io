STREAMER is a machine learning experimenter with a realistic operational stream environment and visualization capabilities.
It facilitates the integration and testing of machine learning algorithms in streaming operational contexts.

Forget about dealing with distributed stream processing engines, STREAMER does it for you!
Experiment with your own use cases or algorithms, and use our API with available algorithms, preprocessing functions and evaluation metrics.

STREAMER is the first open-source academic framework that is scalable and cross-platform.

Architecture

5 modules:

  • M1- Data Stream Ingestion
  • M2- DSPE Processor
  • M3- Learning API + Algs. Adapter.
  • M4- Application & Evaluation.
  • M5- Data Visualization.

Demonstration: ECG unsupervised anomaly detection problem in Stream
